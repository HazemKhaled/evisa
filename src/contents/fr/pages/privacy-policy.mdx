# Politique de Confidentialité

## Introduction

GetTravelVisa.com s'engage à protéger votre vie privée et vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons, stockons et protégeons vos informations.

## Informations que nous collectons

### Informations que vous nous fournissez
- **Informations d'identification** : nom, prénom, date de naissance
- **Coordonnées** : adresse email, numéro de téléphone, adresse postale
- **Documents de voyage** : numéro de passeport, nationalité
- **Informations de voyage** : destination, dates de voyage

### Informations collectées automatiquement
- **Données de navigation** : adresse IP, type de navigateur, pages visitées
- **Cookies et technologies similaires**
- **Données d'utilisation** : comment vous interagissez avec notre service

## Comment nous utilisons vos informations

Nous utilisons vos informations pour :
- **Traitement des demandes de visa** : pour soumettre et suivre vos demandes
- **Communication** : pour vous tenir informé du statut de votre demande
- **Amélioration du service** : pour optimiser notre plateforme
- **Conformité légale** : pour respecter nos obligations réglementaires

## Base légale du traitement

Nous traitons vos données sur la base de :
- **Consentement** : lorsque vous nous donnez votre accord explicite
- **Contrat** : pour l'exécution de nos services
- **Obligation légale** : pour respecter la réglementation
- **Intérêt légitime** : pour améliorer nos services

## Partage des informations

Nous pouvons partager vos informations avec :
- **Consulats et ambassades** : pour le traitement des visas
- **Partenaires de confiance** : prestataires de services de visa
- **Autorités compétentes** : en cas d'obligation légale

## Vos droits

Conformément au RGPD, vous avez le droit de :
- **Accès** : consulter vos données personnelles
- **Rectification** : corriger des informations inexactes
- **Suppression** : demander l'effacement de vos données
- **Portabilité** : récupérer vos données
- **Opposition** : vous opposer au traitement
- **Limitation** : limiter le traitement de vos données

## Sécurité des données

Nous mettons en place des mesures de sécurité appropriées pour protéger vos données contre :
- L'accès non autorisé
- La divulgation
- La modification
- La destruction

## Conservation des données

Nous conservons vos données pendant :
- **Données de demande de visa** : 7 ans après la fin du service
- **Données marketing** : jusqu'à votre désinscription
- **Données de navigation** : 13 mois maximum

## Contact

Pour toute question concernant cette politique de confidentialité ou vos droits, contactez-nous :
- **Email** : privacy@gettravelvisa.com
- **Adresse** : Data Protection Officer, GetTravelVisa.com

Cette politique est effective à compter du 1er janvier 2025.
